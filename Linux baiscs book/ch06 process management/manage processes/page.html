To manage process we use the command `nice`.

- nice takes a number which reflect how nice you are to others. for example, if you use high number means the priority goes down and you are nice to other processes. If using a low number that means you are not nice in sharing the resources with other processes. however the priority goes up.

- the range of nice numbers from -20 to +19

ex: nice -n -10 <process name>

==== change the priority of a running process =====

to change the priority use the command `renice`

ex: renice 16 <process ID>

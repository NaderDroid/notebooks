===

- Port forwarding allow external connections to connect to internal hosts via the router

- Port triggering allow different ports to be triggered when using another port like FTP, the intiation is on port 21 and the response is on port 20

- Enabling DMZ when setting up port forwarding places that device outside the protection area of the router

in this episode we are going to discuss some of the very special ip addresses:

1- private ip address:
a- ip starts with 10.x.x.x
b- ip from 172.16.x.x to 172.31.x.x
c- ip 192.168.x.x

2- loopback addresses:
a- ipv4: 127.0.0.1
b- ipv6: ::1

3- APIPA address: 169.254.x.x

- if you get APIPA address that means the dhcp server is down

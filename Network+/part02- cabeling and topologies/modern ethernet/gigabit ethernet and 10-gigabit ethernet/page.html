there are four gigabit ethernet that are going to be on the exam

1- 1000BaseCX: it uses coaxial cable called Twinax and has a range of 25m between the node and the switch
2- in the fiber optic we have 2 different standards
a- 1000BaseSX multimode for 500m distance
b- 1000BaseLX single mode for distance 5km
3- for UTP, unshielded twisted pair 1000BaseT with distance 100m


on the other hand there is the 10gigabit networks and those have different standards
1- 10GBaseT: if uses cat6 it runs for 55m and if uses cat6a it runs the normal 100m
2- in fiber optic we have the
a- 10GBaseSR multimode
b- 10GBaseLR single mode 1310nm, and runs upto 10km
c- 10GBaseER single mode 1550nm, runs up to 40km

there are many things to consider when connecting switches together
1- loop issues or bridging loops: when connecting switches watch out for loops. This can be solved with STP, spanning tree protocol
2- flood issues: this can solved using new switches by just turning off the port that is flooding the switch or the network
3- missmatch switch: this is rare scenario that switch is not wroking becuase the speed issues
4- dedicated high speed ports: utilize ports that support high speed by connecting them together
5- uplink and crossover: uplink goes to regular with striaght cable or crossover cable from regular to regular ports
6- duplex mismatch: devices need to support full duplex to connect this done automatically but some times needs to be configured manually

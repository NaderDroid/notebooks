There is one term here to be aware of (Convergence)

Convergence means the ability for router to rewrite routing tables to adapt to network faliure and changes and reroute traffic accordingly. Simply means all router tables reflect all routers

- Dynamic routing is the router smartness to route traffic and assign routers routing tables that fit the current network status

- In old days there were two ways to determine the route of the traffic
a- distance vector: this uses the hop count to determine the shortest path
b- link state: this uses what's know as (advertising) which is baisically binging the machines connected to and if any noticed changes in response it will update its routing table and recommend other routers to do so.

=== review points

- Dynamic routing protocols use metrics to determine routes and either distance vector or link state

- Dynamic routing protocols are either EGP, Exterior Gateway Protocol or IGP, Interior Gateway Protocol

- BGP is the EGP used protocol used for inter autnomous system routing
